# Hospital Management System

## Description

The Hospital Management System is a console-based Java application developed to manage patient records efficiently. The system allows users to add new patients, view patient details, search for patients using their ID, delete patient records, and display the total number of patients.

This project is developed using Java and demonstrates Object-Oriented Programming (OOP) concepts such as classes, objects, methods, and collections. It provides a simple and user-friendly interface for managing hospital-related information.

## Features

- Add Patient Records
- View All Patients
- Search Patient by ID
- Delete Patient Records
- Display Total Number of Patients
- Input Validation for User-Friendly Operation

## Technologies Used

- Java
- VS Code
- Git
- GitHub

## How to Run

1. Compile the program:
   javac HospitalManagementSystem.java

2. Run the program:
   java HospitalManagementSystem
